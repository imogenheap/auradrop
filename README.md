# Auradrop

Imogen's personal filing brain. Drop any file → Claude analyses it → files it to Google Drive automatically.

**Live at:** https://imogenheap.github.io/auradrop

## Features
- 🎵 Understands Imogen's world (music, Auracles, WHYDTM, Endlesss, Scout...)
- 📱 Installable as an app on phone, laptop, any device
- 🔌 Works offline — files queue and sync when back online
- 📍 Tracks which device each file came from
- ☁️ All files go to the same Google Drive folders

## Setup
1. Open https://imogenheap.github.io/auradrop
2. Enter your Anthropic API key
3. Click **Connect Google Drive** and sign in
4. Drop files in — done

## Deploy your own
```bash
git clone https://github.com/imogenheap/auradrop
cd auradrop
# Push to GitHub, enable Pages in repo settings → Source: GitHub Actions
```
